# Speech-to-Text Project

## Description
A **Python-based Speech-to-Text (STT) system** that converts audio input into text.  
Built with **PyTorch, Torchaudio, and SpeechBrain**, and optionally uses **Hugging Face pre-trained models** for high accuracy.

---

## Features
- Audio-to-text conversion
- Modular design (preprocessing, model, evaluation, deployment)
- Supports multiple audio formats
- Easy to extend and deploy
